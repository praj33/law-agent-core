# Nyaya Legal Visualization & Dataset Interface

This module exposes Nyaya legal datasets in a structured, UI-consumable format.

## Scope
- Loads jurisdiction-specific legal datasets (IN, UK, UAE)
- Assembles canonical legal context JSON
- Enables frontend visualization of procedures, timelines, and evidence

## Out of Scope
- Legal correctness
- Reinforcement learning
- Domain or jurisdiction inference

## Guarantee
This module does not invent, infer, or modify law.
It only exposes what exists in Nyaya datasets.
